Как работает:
* При добавление предмета, он либо помещается в существующий слот с  существующим предметом, либо помещается в первый свободный слот. Если слотов нет, предмет не добавится. Если в слоте предметов больше чем нужно, то они добавляются в любую свободную ячейку.
* При перемещении предемтов, они могут стакаться, если размер стака в слоте куда перемещают предмет будет максимальным, то остаток вернется в прошлый слот.
Управление:
* ПКМ открывает меню для использования предметов или выброски их. Использование производит заданное действие (в некоторых случаях удаляется предмет). При удалении удаляются все предметы. Закрыть меню можно через клавишу ESC.
* Пермещение происходит зажатием ЛКМ. Если перенести предмет вне инвенторя, то он вернется на свой прошлый слот.
* Создание предметов происходит через меню внизу экрана. У каждого предмета есть две кнопки, Add и Add to slot. Add просто добавляет один предмет в инвентарь. Add to slot добавляет заданное колличество предметов в заданный слот (счет слотов с 0 до n-1). Для Add тоже можно выбрать количество, для этого необохдимо нажать на кнопку рядом с Item Count. Меню создания, не добавляет больше предметов чем максимальный размер стака.
* Для создания нового предмета необходимо выбрать в меню создания StackableItem или Weapon.
* Для настройки размерна инвентаря необходимо выбрать в иерархии Inventory и поменять значение в поле Inventory Size в скрипте Test (делается из инспектора)
